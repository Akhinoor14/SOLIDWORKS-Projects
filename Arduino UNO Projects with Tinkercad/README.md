## âš¡ Tips & Tricks

### ğŸ’¡ For Beginners
1. **Start with Tinkercad** - Simulate before building real circuits
2. **Use Serial.print()** - Debug your code with console output
3. **Test LED polarity** - Long leg (+) to positive, short leg (-) to ground
4. **Common mistake**: Wrong resistor values cause LED damage
5. **PWM pins on UNO**: 3, 5, 6, 9, 10, 11 (check analogWrite())
6. **ADC pins on UNO**: A0-A5 (10-bit, 0-1023 values)

- **Pull-up/Pull-down**: Stabilize floating inputs with 10kÎ© resistors

### Performance Optimization

**For Speed:**
- Use digitalWrite() instead of analogWrite(255)
- Set pinMode() once in setup(), not in loops
- Use bitwise operations for bit manipulation

**For Power (ATtiny85):**
- Disable unused peripherals (ADC, timers)
- Use SLEEP_MODE_PWR_DOWN for minimal current
- Set clock to 1MHz (saves power, slower execution)

**For Accuracy:**
- Take multiple ADC readings and average
- Use voltage reference matching your design
- Calibrate sensors with known values
